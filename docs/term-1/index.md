# Term 1: Foundational
Welcome to your first term at the Academy. In this term, we focus entirely on the **mental models** and **core tools** required to be a Data Engineer.

We will not be building complex cloud pipelines yet. Instead, we are learning the "Language of Data." By the end of this term, you will understand how data moves, how to control your computer via the command line, and how to write the two most important languages in data: Python and SQL.

### [103 - SQL for Data Engineering (Basics)](103-sql/01-select.md)
* **Focus:** Database Querying
* **Goal:** Learn to talk to the database.
* **Key Topics:**
    * `SELECT` & `FROM`: Getting data out.
    * `JOINS`: Combining different datasets (The #1 most important skill).
    * Aggregations: Summarizing data for the business.

### [104 - Developer Toolkit](104-toolkit/01-terminal.md)
* **Focus:** Environment & Version Control
* **Goal:** Stop clicking, start engineering.
* **Why this matters:** Real engineers don't drag-and-drop files. They use the **Command Line**. You will also learn **Git**, which saves your code history and stops you from losing work.
